The project is an online community runing on Android devices
User can post own messages (text only), and read, reply, and vote (up vote or down vote) for existing messages
All messages are stored in a google firebase realtime database, and the app monitors and make changes on the database

To edit or test the app
Download all files, decompress the rar pack, and import the project with Android studio.
